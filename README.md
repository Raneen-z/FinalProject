## Set up  
### Prerequisites
- VS Code

#### in the terminal
``` 
    git clone https://github.com/Raneen-z/FinalProject.git
    cd zelda-botw
    npm i 
```
#### Run JSON Server 
``` 
json-server --watch db.json 
```
it will run on port 3000

#### Start React App
in another port run
```
 npm start
  ```
